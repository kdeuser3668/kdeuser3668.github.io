Course Reflection

My Learning Experience

Throughout this course, I have learned many foundational concepts about web design and development. Creating webpages using HTML and CSS has helped me understand how websites are structured and styled. At first, learning the syntax and structure of HTML felt overwhelming, but with practice it became much easier to understand how the different elements work together.

One of the most valuable skills I developed was learning how to organize content using semantic elements such as headers, navigation bars, and main content areas. This made my pages feel more structured and easier to read. I also learned how CSS can control the appearance of a website, allowing designers to change colors, spacing, and layout in a consistent way.

Another important concept I learned was how to create navigation between pages. Linking multiple pages together makes a website feel complete and functional. This project helped me understand how users move through a site and why clear navigation is important for usability.

Overall, this course has helped me feel more confident in my ability to create a basic website. I now understand how HTML provides the structure and CSS controls the design. These skills will be useful in future classes and projects where web development or digital content creation is required.
